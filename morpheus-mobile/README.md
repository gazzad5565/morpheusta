# morpheus-mobile

Mobile rep app (Next.js PWA) — part of the Morpheus Field Operations Suite.

**This folder doesn't have its own docs.** Project documentation lives in the top-level README:

→ **[../README.md](../README.md)**

That covers architecture, setup, database, deployment, and how to onboard a new machine.

---

Quick reference (assumes you've already followed the setup in the top-level README):

```bash
npm run dev      # local dev server, http://localhost:3000
npm run build    # production build
npm run start    # serve the production build
```
